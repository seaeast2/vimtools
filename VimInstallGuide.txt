1. Setup Vundle:
  $ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

2. Configure bundles:
  .vimrc

3. Install configured bundles:
  Launch [vim], run [:PluginInstall] (or [vim +Plugininstall +qall] for CLI lovers)
